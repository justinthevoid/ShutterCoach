# Contributing to ShutterCoach

Thanks for your interest in ShutterCoach! While the source code is not open-source, your feedback is incredibly valuable and directly shapes the app's development.

## How to Contribute

### Report a Bug

Found something broken? [Open a bug report](https://github.com/justinthevoid/ShutterCoach/issues/new?template=bug-report.yml) with as much detail as you can — your app version, iOS version, steps to reproduce, and screenshots all help us fix things faster.

### Request a Feature

Have an idea for a new feature? [Submit a feature request](https://github.com/justinthevoid/ShutterCoach/issues/new?template=feature-request.yml) and tell us what you'd like and why it would help your photography workflow.

### Share General Feedback

Love something about the app? Have a small suggestion? Just want to say hi? [Send us feedback](https://github.com/justinthevoid/ShutterCoach/issues/new?template=general-feedback.yml) — we read every message.

### Join the Discussion

For open-ended questions, photography tips, or community conversation, head over to [GitHub Discussions](https://github.com/justinthevoid/ShutterCoach/discussions). It's the best place for topics that aren't bugs or feature requests.

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md). We're building a welcoming space for photographers of all levels.

## Note on Source Code

ShutterCoach is a closed-source application. This repository is for community engagement, issue tracking, and public documentation — not for code contributions. We appreciate your understanding!
