# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.1     | Yes       |
| < 1.1   | No        |

## Reporting a Vulnerability

If you discover a security issue in ShutterCoach, please report it responsibly.

**Email:** support@shuttercoach.app

**What to expect:**
- We will acknowledge your report within 48 hours
- We will work with you to understand and resolve the issue
- We ask that you do not publicly disclose the issue until a fix is available

**Recognition:** With your permission, we will credit researchers who report valid issues in the CHANGELOG.

Thank you for helping keep ShutterCoach and its users safe.
